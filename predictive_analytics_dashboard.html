Model pipeline

1
Data ingestion & cleaning — parse timestamps, handle missing values via interpolation, remove outliers using IQR fencing (Q1 − 1.5×IQR, Q3 + 1.5×IQR).
2
Feature engineering — extract lag features, rolling means, and seasonal dummies. Normalize with min-max scaling before fitting.
3
Model training — fit on 80% training split. Hyperparameters tuned via time-series cross-validation (walk-forward validation).
4
Evaluation — measure MAE, RMSE, and R² on 20% held-out test set. Residuals checked for autocorrelation (Ljung-Box test).
5
Forecast & uncertainty — generate point forecasts + 95% confidence intervals via bootstrapped residuals. Widen interval with horizon.
